# Inventory Management System

## Project Description

This project is an Inventory Management System developed using Django Framework and Bootstrap 5.

ប្រព័ន្ធនេះត្រូវបានបង្កើតឡើងសម្រាប់គ្រប់គ្រង:
- Products
- Categories
- Suppliers
- Stock

It also includes an admin dashboard with authentication system for secure management.

---

# Features

## Authentication
- Login system
- Logout system
- Register admin account
- Protected admin pages

## Product Management
- Add product
- Edit product
- Delete product
- Product list

## Category Management
- Add category
- View category list

## Supplier Management
- Add supplier
- View supplier list

## Stock Management
- Update product stock
- Increase or decrease quantity

# Installation

## Clone Repository

```bash
git clone https://github.com/likvan21-lab/finalassignment.git
```

## Go To Project Folder

```bash
cd finalassignment
```

## Install Django

```bash
pip install django
```

## Run Server

```bash
python manage.py runserver
```
